SoC Ideas

Here is a list of ideas for the Google Summer of Code. Feel free to consider any of these ideas or a project of your own choosing.

  • Installer for OS: A graphical installer written using a toolkit currently ported to MidnightBSD. Could be Gtk, QT, etc. The system must be easy to use with minimal questions for the user. Possible examples might be the OS X installer or PC-BSD's installer.
  • A system preferences management system: This would allow the user to customize what services are running, and manage network settings.
  • Software Update: Develop a system using the new mport tool to update operating system components as packages.
  • Port ZFS from FreeBSD/Solaris.
  • Veryify the SUS and POSIX compliance of system utilities and interfaces in the system. Report on these and offer suggestions to fix them
  • Write software applications for desktop users that run on MidnightBSD using GNUstep, Gtk or QT
  • Write a firewall management application (graphical) for PF or IPFW.
  • Port Gjournal from FreeBSD.
  • Port wireless drivers from the other BSDs.
