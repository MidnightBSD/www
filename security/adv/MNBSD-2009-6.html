MNBSD-2009-6: Local memory disclosure via anonymous pipes

Severity: Unknown

Affected Package: kernel

Summary: Local memory disclosure via anonymous pipes

Description

Unprivileged processes could read pages of memory belonging to other processes through anonymous pipes, resulting in local information disclosure. The fix was applied to 0.2.1 (patch level p9) and 0.3-CURRENT. This corresponds to FreeBSD-SA-09:09.pipe, announced the same day.

Affected Versions

kernel
