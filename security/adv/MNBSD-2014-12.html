MNBSD-2014-12: Kernel memory leak in sandboxed namei path lookup

Severity: Unknown

Affected Package: kernel

Summary: Kernel memory leak in sandboxed namei path lookup

Description

When a Capsicum-sandboxed process performed a namei lookup of a nonexistent path name, the kernel could leak memory. This could lead to resource exhaustion and disclosure of kernel memory.

Affected Versions

kernel
