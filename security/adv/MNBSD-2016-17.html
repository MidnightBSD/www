MNBSD-2016-17: Buffer overflow in link_ntoa(3)

Severity: Unknown

Affected Package: libc

Summary: Buffer overflow in link_ntoa(3)

Description

Improper bounds checking of the output buffer in the link_ntoa() function in linkaddr.c of libc could allow reading or writing beyond the buffer when given crafted input. Exploitable applications are considered very unlikely to exist in practice. Fixed in the 0.8.5 release.

Affected Versions

libc
