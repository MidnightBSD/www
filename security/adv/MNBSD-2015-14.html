MNBSD-2015-14: SCTP ICMPv6 input validation kernel panic

Severity: Unknown

Affected Package: kernel

Summary: SCTP ICMPv6 input validation kernel panic

Description

A lack of proper input checks in the ICMPv6 processing of the SCTP stack could lead to either a failed kernel assertion or a NULL pointer dereference. In either case a kernel panic follows, allowing a remote attacker to cause a denial of service.

Affected Versions

kernel
