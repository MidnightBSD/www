MNBSD-2021-6: Linux compatibility layer futex(2) denial of service

Severity: Unknown

Affected Package: kernel

Summary: Linux compatibility layer futex(2) denial of service

Description

A programming error in the Linux compatibility layer futex(2) system call, where an unprivileged local attacker could specify a negative wake or requeue value causing a signed integer overflow, might allow attackers to cause a denial of service. Systems not using the Linux binary compatibility layer are not affected.

Affected Versions

kernel
