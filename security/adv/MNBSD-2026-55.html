MNBSD-2026-55: unbounded name compression causes degradation of service

Severity: Unknown

Affected Package: unbound

Summary: unbounded name compression causes degradation of service

Description

In certain cases unbound performs unbounded DNS name compression, which can cause a degradation of service (Denial of Service). Reported by Qifan Zhang, Palo Alto Networks. Fixed in unbound 1.25.1.

Affected Versions

unbound
