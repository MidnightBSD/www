MNBSD-2014-18: sshd libpthread symbol shadowing denial of service

Severity: Unknown

Affected Package: openssh

Summary: sshd libpthread symbol shadowing denial of service

Description

Due to incorrect library ordering while linking sshd(8), symbols in the C library that are shadowed by the POSIX thread library may not be resolved correctly. This could cause a denial of service against connecting clients.

Affected Versions

openssh
