MNBSD-2014-4: TCP reassembly denial of service and memory disclosure

Severity: Unknown

Affected Package: kernel

Summary: TCP reassembly denial of service and memory disclosure

Description

A TCP reassembly bug could result in a denial of service against the system, and it may also be possible to obtain portions of kernel memory. A remote attacker could exploit this by sending crafted TCP segments.

Affected Versions

kernel
