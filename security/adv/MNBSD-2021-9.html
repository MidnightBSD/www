MNBSD-2021-9: Stack buffer overflow in the ggatec(8) GEOM Gate client

Severity: Unknown

Affected Package: ggatec

Summary: Stack buffer overflow in the ggatec(8) GEOM Gate client

Description

The ggatec(8) GEOM Gate client daemon did not validate the size of a response before writing it to a fixed-size buffer. A malicious server could send crafted responses to overflow the stack of ggatec(8) and potentially execute arbitrary code.

Affected Versions

ggatec
