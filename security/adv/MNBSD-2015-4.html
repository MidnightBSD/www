MNBSD-2015-4: OpenSSL security update to 0.9.8zg (multiple vulnerabilities)

Severity: Unknown

Affected Package: openssl

Summary: OpenSSL security update to 0.9.8zg (multiple vulnerabilities)

Description

OpenSSL was updated to version 0.9.8zg to address multiple vulnerabilities, including the Logjam DH downgrade weakness and several denial-of-service and memory issues. Impacts range from denial of service to man-in-the-middle downgrade of TLS connections.

Affected Versions

openssl
