MNBSD-2026-53: possible cache poisoning attack while following delegation

Severity: Unknown

Affected Package: unbound

Summary: possible cache poisoning attack while following delegation

A flaw while following delegations enables a possible cache poisoning attack. Reported by TaoFei Guo (Peking University), Yang Luo and JianJun Chen (Tsinghua University).
