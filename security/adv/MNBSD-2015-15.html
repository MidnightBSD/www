MNBSD-2015-15: TCP MD5 signature handling denial of service

Severity: Unknown

Affected Package: kernel

Summary: TCP MD5 signature handling denial of service

Description

A programming error in processing a TCP connection using both the TCP_MD5SIG and TCP_NOOPT socket options could lead to a kernel crash. An attacker or local application able to establish such a connection could panic the system, resulting in a denial of service.

Affected Versions

kernel
