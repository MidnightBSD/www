MNBSD-2008-9: ftpd flaw allowing unauthorized access, network exploitable

Severity: Unknown

Affected Package: ftpd

Summary: ftpd flaw allowing unauthorized access, network exploitable

Description

A vulnerability in ftpd could allow unauthorized access and was network exploitable, affecting all versions of MidnightBSD. Users were advised to update via cvs to RELENG_0_2 or apply the provided patch and rebuild ftpd. This corresponds to FreeBSD-SA-08:12.ftpd.

Affected Versions

ftpd
