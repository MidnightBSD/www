MNBSD-2021-3: Xen grant-mapping error handling resource leak

Severity: Unknown

Affected Package: kernel

Summary: Xen grant-mapping error handling resource leak

Description

Improper error handling in the Xen grant-mapping operations of the blkfront/netfront paths could allow a malicious or buggy frontend driver to cause resource leaks in the domain running the corresponding backend driver, and to unmap memory incorrectly when errors occur.

Affected Versions

kernel
