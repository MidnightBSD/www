MNBSD-2015-2: IGMPv3 integer overflow denial of service

Severity: Unknown

Affected Package: kernel

Summary: IGMPv3 integer overflow denial of service

Description

An integer overflow in computing the size of the IGMPv3 data buffer could result in a buffer that is too small for the requested operation. This could be exploited to cause a denial of service.

Affected Versions

kernel
