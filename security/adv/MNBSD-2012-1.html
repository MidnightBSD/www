MNBSD-2012-1: named crash from zero-length resource record data

Severity: Unknown

Affected Package: bind

Summary: named crash from zero-length resource record data

Description

A defect in BIND's handling of resource records allows a request with zero-length rdata to crash named(8), resulting in a denial of service or possible disclosure of information. Systems running the BIND name server are affected.

Affected Versions

bind
