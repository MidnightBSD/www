MNBSD-2014-19: Denial of service in file(1) and libmagic

Severity: Unknown

Affected Package: file

Summary: Denial of service in file(1) and libmagic

Description

Multiple vulnerabilities in file(1) and libmagic(3) could allow an attacker to cause a denial of service in any program that uses libmagic, for example via a crafted ELF or other input processed by the magic parser. This affects services that identify untrusted file content.

Affected Versions
