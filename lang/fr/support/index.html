MidnightBSD: The BSD pour Tous

Aide & Support

Pendant les étapes de planification initiales, la période de support de MidnightBSD est limitée. Il vous est recommandé de lire le manuel FreeBSD®.

Vous pouvez aussi consulter la documentation officielle de MidnightBSD pour installer et utiliser le système mports.

La page Communauté a aussi des liens vers des ressources utiles, tel que la chatroom #MidnightBSD IRC.

Toutes les failles de sécurité de FreeBSD® ont été corrigées dans MidnightBSD en Octobre 2015.

Si vous devez rapporter une faille de sécurité dans MidnightBSD, contactez le responsable de la sécurité à gro.dsbthgindim@ytiruces.

Envoyez un e-mail à support@midnightbsd.org pour toutes questions techniques que vous pourriez avoir. Vous pouvez également utiliser la liste de diffusion.
