MidnightBSD: Le Système BSD pour Tous

Bienvenue chez MidnightBSD

MidnightBSD est un système d'exploitation BSD développé pour les utilisateurs d'un PC de bureau. MidnightBSD inclut tous les logiciels dont vous avez besoin au quotidien — email, navigation internet, traitement de texte, jeux, et bien plus encore.

Avec une petite communauté de développeurs dévoués, MidnightBSD s'efforce de créer un système d'exploitation facile d'utilisation pour que tout le monde puisse s'en servir, librement. MidnightBSD est disponible pour les architectures x86, AMD64 et aussi les Machines Virtuelles.
