MidnightBSD: Le BSD pour Tous

Idées SoC

Ci-dessous une liste d'idées pour le Google Summer of Code. N'hésitez pas à choisir une de ces idées ou un de ses projets.

Difficulté Connaissances Utiles
Installateur pour l'OS: Un installateur graphique écrit en utilisant un toolkit actuellement supporté par MidnightBSD. Cela peut être Gtk, Qt, etc. Le système doit être facile à utiliser avec le minimum de questions pour l'utilisateur. Des exemples possibles peuvent être l'installateur d'OS X or l'installateur de PC-BSD. Moyen Être familier avec les processus d'installation; Background in Gtk, QT, etc.
Un système de gestion des préfèrences système: Cela permettrait à l'utilisateur de personnaliser quels services sont en train de tourner, et de gérer les paramétres du réseau. Moyen Background dans Gtk, QT, etc.
Mise à jour logiciel: Développer un système utilisant le nouvel outil mport pour mettre à jour les composants du système d'exploitation comme les paquets. Moyen C, Background dans Gtk ou autre toolkit GUI
Veryify the SUS and POSIX compliance of system utilities and interfaces in the system. Report on these and offer suggestions to fix them Moyen C; Good BSD background; Familiar with SUS and POSIX standards.
Write software applications for desktop users that run on MidnightBSD using GNUstep, Gtk or QT De Facile à Moyen Familiar with GUI toolkit; C, C++ or Objective C
Write a firewall management application (graphical) for PF or IPFW. Moyen C ou C++
Port wireless drivers from the other BSDs. De Moyen à Hard C ou C++
