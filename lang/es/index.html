MidnightBSD: El BSD Para Todos

Bienvenido a MidnightBSD

MidnightBSD es un nuevo sistema operativo BSD-derivado desarrollado con los usuarios de escritorio en mente. Incluye todo el software que esperarías para tus tareas diarias — correo electrónico, navegación web, procesamiento de textos, juegos, y mucho más.

Con una pequeña comunidad de desarrolladores dedicados, MidnightBSD se esfuerza en crear un sistema operativo fácil de usar para que todos puedan disfrutar de él libremente Disponible para x86, AMD64 y como Máquinas Virtuales.
