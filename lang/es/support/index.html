MidnightBSD: El BSD Para Todos

Ayuda y Soporte

Durante las etapas de planificación inicial, el soporte para MidnightBSD es limitado. Es recomendable que veas el Manual de FreeBSD®.

Puedes ver también la documentación oficial de MidnightBSD para ayuda instalando y usando el sistema de mports.

La página de la Comunidad también tiene enlaces a recursos útiles, tal como la sala de chat IRC #MidnightBSD.

Todos los avisos de seguridad de FreeBSD® han sido parcheados en MidnightBSD a partir de Octubre de 2015.

Si necesitas reportar un asunto de seguridad con MidnightBSD, contacta al oficial de seguridad a gro.dsbthgindim@ytiruces.

Dirección de correo electrónico support@midnightbsd.org con cualquier pregunta de soporte que puedas tener. Puedes usar también nuestras listas de correo electrónico.
