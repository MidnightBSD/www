MidnightBSD: El BSD Para Todos

SoC Ideas

Aquí hay una lista de ideas para el Google Summer of Code. Siéntete libre de considerar cualquiera de estas ideas o un proyecto de tu propia elección.

Dificultad Conocimiento Útil
Instalador para SO: Un instalador gráfico escrito usando un juego de herramientas actualmente portado a MidnightBSD. Puede ser Gtk, QT, etc. El sistema debe ser fácil de usar con lás mínimas dudas para el usuario. Posibles ejemplos pueden ser el instalador de OS X o el de TrueOS®. Media Familiarizado con el proceso de instalación; Fondo en Gtk, QT, etc.
Un sistema de gestión de preferencias del SO: Esto permitiría al usuario personalizar que servicios están corriendo, y administrar los ajustes de red. Media Experimentado en Gtk, QT, etc.
Actualización de Software: Desarrollar un sistema usando la nueva herramienta mport para actualizar los componentes del SO como paquetes. Media C, Experimentado en Gtk u otro juego de herramientas GUI
Verificar el cumplimiento de SUS y POSIX de las utilidades del sistema e interfaces en el sistema. Reportar sobre estos y ofrecer sugerencias para solucionarlos. Media C; Bien Experimentado BSD; Familiarizado con los estandares SUS y POSIX.
Escribir aplicaciones de software para usuarios de escritorio que corren MidnightBSD usando GNUstep, Gtk or QT Fácil a Media Familiarizado con el juego de herramientas GUI; C, C++ u Objective C
Escribir una aplicación de administración de cortafuegos (gráfica) para PF o IPFW. Media C o C++
Portar controladores inalámbricos desde otros BSDs. Media a Alta C or C++
